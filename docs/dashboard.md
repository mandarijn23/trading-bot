# Dashboard

The dashboard is a lightweight operations view for paper trading and early validation.

## Commands

```bash
python dashboard.py
```

## What It Displays

- closed trade summary
- win rate and profit factor
- drawdown metrics
- results by symbol
- results by day
- recent trades

## Data Sources

- trade history CSV generated by the bots
- dashboard output in the terminal

## Practical Use

Use the dashboard to confirm that the system is producing trades and staying within the risk profile you expect before changing strategy settings.
